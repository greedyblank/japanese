# 《分類語彙表》词条标签覆盖

## 目标

离线重建 192 条种子词汇的 `tags`，不调用 LLM 或网络服务。

## 标签模型

每条词汇恰好使用一个国立国語研究所《分類語彙表》严格的「類・部門」根标签：

`体・関係`、`体・主体`、`体・活動`、`体・生産物`、`体・自然`、`用・関係`、`用・活動`、`用・自然`、`相・関係`、`相・活動`、`相・自然`、`その他`。

词性、活用等仍由现有词条字段呈现，不能写入 `tags`。

## 映射规则

使用一个本地、按词条 `kanji` 精确匹配的映射表覆盖所有 192 项；每项只映射至一个上述标签。映射表与种子数据同存，便于审阅与重复运行。

无法归入词汇三大类（体、用、相）的拟声拟态、语法性项目等，归入 `その他`。

## 验收

- 192 条词汇均有且仅有一个标签。
- 所有标签都属于上述 12 桶。
- 不调用 LLM、`fetch` 或外部服务。
- 自动化测试覆盖完整性、合法性及典型词汇分类。
